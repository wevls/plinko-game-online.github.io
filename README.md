# Versflip Landing Page

A Flask-powered landing page for Versflip with a pink and purple visual theme inspired by the provided design. The page highlights trusted, server-driven flips with quick calls to action and featured game cards.

## Getting started
1. Create and activate a virtual environment (recommended).
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the development server:
   ```bash
   python app.py
   ```
4. Open http://localhost:5000 to view the Versflip experience.

## Design notes
- Gradient pink and purple palette across buttons, cards, and charts.
- Versflip logo badge built from text for an easily brandable header.
- Responsive grid layout for the hero section, chart preview, and featured game cards.
